## Telegram HTML format

- `<b>` for bold text
- `<i>` for italic text
- `<u>` for underlined text
- `<s>` for strikethrough text
- `<tg-spoiler>` for spoiler text
- `<a href="URL">` for inline links
- `<a href="tg://user?id=USER_ID">` for mentioning a user by their ID
